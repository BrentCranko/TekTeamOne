The store\+\_\+old\+\_\+sp pointer points to a location where the SP of the current execution context will be saved. The next\+\_\+sp contains the SP (not its address!) of the next executuion context (which was previously saved in this manner).

The initial stack of a context must be constructed to be compatible with the switching code (file swicth\+\_\+to.\+asm).
